# 新年答案之书

一个互动式答案之书网页应用。

## 功能特点

- **动画效果**：添加了背景浮动、标题闪烁、按钮发光等动画
- **响应式设计**：支持桌面和移动设备访问
- **局域网共享**：可通过局域网让其他设备访问

## 快速开始

### 方式一：直接打开文件

1. 直接在浏览器中打开 `index.html` 文件
2. 输入你的问题，点击“获取答案”按钮

### 方式二：局域网服务器（推荐）

使用服务器可以让同一局域网内的其他设备（如手机、平板）访问网页。

1. 确保你的电脑已安装 Python 3
2. 在项目目录下运行服务器：

```bash
python3 server.py
```

3. 服务器启动后，会显示访问地址：
   - 本地访问：http://localhost:8000/index.html
   - 局域网访问：http://你的IP地址:8000/index.html

4. 在其他设备上打开浏览器，输入局域网访问地址即可访问

## 使用说明

1. **输入问题**：在输入框中输入你的问题
2. **获取答案**：点击“获取答案”按钮或按回车键
3. **观看动画**：书会摇晃并显示答案
4. **重新提问**：可随时输入新问题并再次查询

## 项目文件

- `index.html` - 主页面文件（包含HTML、CSS和JavaScript）
- `server.py` - 局域网服务器脚本
- `README.md` - 项目说明文档

## 技术实现

- **HTML5** - 页面结构
- **CSS3** - 样式设计和动画
- **JavaScript** - 交互逻辑
- **Python** - 局域网服务器

## 兼容性

- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 响应式设计，适配各种屏幕尺寸
- 无需网络连接（本地文件模式）
