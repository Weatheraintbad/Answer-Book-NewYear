# 🐴 马年答案之书

一个充满马年新年氛围的互动式答案查询网页应用。

## ✨ 功能特点

- 🎨 **新年主题设计**：采用红色和白色为主色调，充满节日氛围
- 🐴 **马年元素**：包含马年主题的动画和装饰
- 💫 **动画效果**：添加了背景浮动、标题闪烁、按钮发光等动画
- 📱 **响应式设计**：支持桌面和移动设备访问
- 🌐 **局域网共享**：可通过局域网让其他设备访问

## 🚀 快速开始

### 方式一：直接打开文件

1. 直接在浏览器中打开 `答案之书新春版.html` 文件
2. 输入你的问题，点击"领取新年答案"按钮
3. 享受马年主题的互动体验

### 方式二：局域网服务器（推荐）

使用服务器可以让同一局域网内的其他设备（如手机、平板）访问网页。

1. 确保你的电脑已安装 Python 3
2. 在项目目录下运行服务器：

```bash
python3 server.py
```

3. 服务器启动后，会显示访问地址：
   - 本地访问：http://localhost:8000/答案之书新春版.html
   - 局域网访问：http://你的IP地址:8000/答案之书新春版.html

4. 在其他设备上打开浏览器，输入局域网访问地址即可访问

## 🎯 使用说明

1. **输入问题**：在输入框中输入你的问题
2. **获取答案**：点击"领取新年答案"按钮或按回车键
3. **观看动画**：书会摇晃并显示答案
4. **重新提问**：可随时输入新问题并再次查询

## 🎨 设计特色

### 主题色彩
- **主色**：红色 (#d32f2f) - 象征喜庆和吉祥
- **辅助色**：金色 (#ffeb3b) - 象征财富和好运
- **背景**：白色渐变背景，带有红色和金色点缀

### 动画效果
- **背景浮动**：淡红色和金色圆点的重复图案
- **标题闪烁**：主标题周期性缩放动画
- **书籍发光**：书籍标题有金色光晕效果
- **按钮特效**：鼠标悬停时显示星星效果
- **答案弹出**：答案显示时有弹出动画
